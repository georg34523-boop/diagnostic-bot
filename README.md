# 🏥 Diagnostic Bot — Инструкция по установке

## Что мы создали

1. **Telegram-бот** — принимает сообщения и фото от клиентов
2. **API-сервер** — связывает бота с веб-панелью
3. **Веб-панель** — CRM для эксперта с чатами, воронкой, напоминаниями

---

## Шаг 1: Настройка Supabase

### 1.1 Создание таблиц

1. Открой свой проект: https://supabase.com/dashboard
2. Перейди в **SQL Editor**
3. Скопируй содержимое файла `sql/init.sql`
4. Нажми **Run**

### 1.2 Создание Storage bucket

1. Перейди в **Storage** (в левом меню)
2. Нажми **New bucket**
3. Имя: `diagnostic-files`
4. Поставь галочку **Public bucket**
5. Нажми **Create bucket**

### 1.3 Получение Service Key (для бота)

1. Перейди в **Settings → API**
2. Скопируй **service_role key** (НЕ anon key!)
3. Замени в файле `bot/.env` значение `SUPABASE_KEY` на этот ключ

> ⚠️ Service key даёт полный доступ к базе. Никому не показывай!

---

## Шаг 2: Запуск бота локально (для теста)

### 2.1 Установка Python

Если нет Python, скачай с https://python.org (версия 3.10+)

### 2.2 Установка зависимостей

```bash
cd bot
pip install -r requirements.txt
```

### 2.3 Запуск бота

```bash
python main.py
```

Если видишь `Запуск бота...` — бот работает!

### 2.4 Запуск API (в отдельном терминале)

```bash
pip install fastapi uvicorn
python api.py
```

API доступен на http://localhost:8000

## Шаг 3: Запуск веб-панели

### Вариант А: Локально через Vite

```bash
# Установи Node.js если нет: https://nodejs.org

cd web
npm create vite@latest . -- --template react
npm install @supabase/supabase-js
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Замени `src/App.jsx` на наш `App.jsx`

Добавь в `tailwind.config.js`:
```js
content: ["./src/**/*.{js,jsx}"]
```

Добавь в `src/index.css`:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Запусти:
```bash
npm run dev
```

## Шаг 4: Деплой в продакшен

### Бот + API → Railway.app

1. Зарегистрируйся на https://railway.app
2. Создай новый проект из GitHub
3. Добавь переменные окружения:
   - `BOT_TOKEN`
   - `SUPABASE_URL`
   - `SUPABASE_KEY`

### Веб-панель → Vercel

1. Зарегистрируйся на https://vercel.com
2. Подключи репозиторий
3. Vercel сам определит Vite и задеплоит

## Как пользоваться

### Добавление клиента (кто оплатил)

1. Открой веб-панель
2. Перейди в **👥 Доступ**
3. Введи @username клиента в Telegram
4. Нажми **Добавить**

Теперь клиент может писать боту!

### Работа с клиентами

1. Клиент пишет боту → появляется в списке чатов
2. Эксперт видит сообщение, отвечает
3. Меняет статус по воронке
4. Добавляет заметки и напоминания

### Воронка

- **Новый** — только написал
- **Диагностика запланирована** — договорились о времени
- **Диагностика проведена** — разобрали фото
- **Звонок запланирован** — договорились о созвоне
- **Звонок проведён** — финал
